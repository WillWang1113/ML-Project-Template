# ML-Project-Template
Python template for machine learning

## Usage
Before you begin a new ML project, create a repo from this template.

Hit the button **Use this template** on the webpage to create a repo.

## Stuctrue
```
./
├── configs
│   ├── json_config.json
│   └── yaml_config.yaml
├── LICENSE
├── notebooks
│   └── my_notebook.ipynb
├── README.md
├── run.sh
├── scripts
│   └── my_experiments.py
├── setup.py
├── src
│   └── __init__.py
└── tests
```
